07 C语言/replace

本文主要针对c++中常用replace函数用法给出样例程序

  1. int main()
  2. {
  3. string line = “this@ is@ a test string!”;
  4. line = line.replace(line.find(“@”), 1, “”);
  5. cout << line << endl;
  6. return 0;
  7. }
