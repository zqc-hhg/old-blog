03 Linux/命令/基本命令/文本文件编辑命令

命令 作用 格式 备注
cat 查看纯文本文件（内容较少的） cat [选项] 文件
more 查看纯文本文件（内容较多的） more [选项]文件
head 查看纯文本文档的前N行 head [选项] 文件
tail 查看纯文本文档的后N行或持续刷新内容 tail [选项] 文件
tr 替换文本文件中的字符 tr [原始字符] [目标字符]
wc 统计指定文本的行数、字数、字节数 wc [参数] 文本
stat 查看文件的具体存储信息和时间等信息 stat 文件
cut 按“列”提取文本字符 cut [参数] 文本
diff 比较多个文本文件的差异 diff [参数] 文件
